AI Settings Optional

Paste an OpenAI API key here to enable the "✨ Generate description" button in the risk edit panel. The key is stored only in your browser's localStorage — it never leaves your device. Do not use this if the file is hosted publicly or shared widely.

Admin PIN

Import Risk Data

Bulk add or update risks from an Excel or CSV file. Matching Risk IDs are updated; new IDs are added to the register.
Expected column format (click to expand)
Risk IDOptional — leave blank to auto-assign the next available ID.
TitleRequired — short risk name.
DescriptionOptional — full risk description (also accepts "Final Description").
Group FamilyOptional — should match an existing Group Family, managed above.
Risk UniverseOptional — use the "CODE_Label" format, e.g. C7_Cybersecurity attacks and system compromise.
BUOptional — Business Unit code, e.g. FRA, IND, USA, managed above.
OwnerOptional — risk owner's name.
StatusOptional — defaults to "Ongoing" if blank, managed above.
Likelihood / Impact / ControlOptional, default to 2 — accepts a number 1–4 or label text (e.g. "Rare", "High", "Defined").
Column headers are flexible and can appear in any order. Tip: the Risk Register's CSV export uses these exact column names — edit it in Excel and re-import to round-trip safely.
